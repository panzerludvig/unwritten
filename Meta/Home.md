# Home

[[Home]] is the front page of the vault — the first note Obsidian opens. Its purpose is human navigation: a quick orientation to the project and links to the main sections. It is not a briefing document for Claude; use the [[Codex]] for that.

---

## What belongs in Home

- A one-line project description
- Quick links to the top-level notes of each major section
- Repository location (GitHub URL and local path)
- Current project status (stage, start date)

Home should stay short. It is a doorway, not a document.

---

## What does not belong in Home

- Detail about any topic — link to the relevant note instead
- Instructions or standards — those live in the Codex
- History or decisions — those live in the Dev Log

---

## Keeping it current

Update the Status section when the project stage changes. Add a quick link when a new top-level section is created. Nothing else should need regular maintenance.
